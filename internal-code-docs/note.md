# Note

The Fault×32 project is split across multiple repositories. Each repository targets a different environment.

This documentation covers the [Fault×32 game core](https://github.com/Some-Dev-Team/Fault32) repository. It contains scripts for the Roblox environment.

Other repositories remain private for security reasons.

This documentation was made public to allow players and other researchers to understand how our simulation works internally, learn from it and potentially point our some bugs and inconsistencies.

If you have any other questions reach out to our community ^\_^ you can find links in main [..](../ "mention") page.
